# Franchiseen

## Overview
This is the **demo release** of **Franchiseen**, a blockchain-powered franchise investment and management platform.  
The demo is built specifically for the **marketing team** to showcase the platform’s core flows with **Phantom Wallet integration**.

Founder: **Shawaz Sharif** (UI/UX Designer, building Franchiseen from scratch).  
Future: Once we onboard our first **5 brands**, we will launch the **main wallet integration (Phantom)** for production and expand further.

---

## Demo Setup

### 1. Web (Phantom Wallet)
- The **web app** acts as the **main Phantom wallet integration point**.
- Brands and investors can **connect their Phantom wallets** for authentication and demo transactions.
- Used to showcase:
  - **Wallet connection flow**
  - **Franchise token listing (demo)**
  - **Basic transaction simulation**

### 2. Mobile (React Native App)
- The **mobile app** is the demo version of **Franchiseen**.
- Features in the demo:
  - User login (demo)
  - Explore franchises
  - Create/join franchise (demo flow)
  - Phantom wallet connect (via deep link / dev mode)

---

## Demo Scope
⚠️ This is **NOT production ready**.  
It is **for demo purposes only**:
- No real investments yet
- No fiat ramps (USDC only in demo)
- Payouts are simulated

## Tech Stack
- **Blockchain:** Solana (Phantom Wallet)  
- **Frontend (Web):** Next.js + Phantom Wallet Adapter

- **Backend:** Convex (temporary), will upgrade after CTO hire
